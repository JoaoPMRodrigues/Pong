# Pong em Python (PPLAY)

Implementação do clássico jogo Pong, desenvolvida como parte da disciplina de Laboratório de Jogos.

## Sobre o projeto

Este projeto recria o jogo Pong, um dos primeiros jogos eletrônicos da história, com foco em:

1. Lógica de jogo
2. Detecção de colisão
3. Controle de entrada do usuário
4. Estruturação de código

O objetivo foi consolidar conceitos fundamentais de programação aplicados ao desenvolvimento de jogos.
